# C# Exercise Series

This repository consists of practical exercises designed to reinforce C# programming skills from basic to advanced levels.

---

## Exercise Categories

| Range   | Topic                              | Description                                                               |
| ------- | ---------------------------------- | ------------------------------------------------------------------------- |
| 1 – 10  | Basic Input/Output & Arithmetic    | Console input/output, addition, subtraction, multiplication, division     |
| 11 – 20 | Loops & Simple Patterns            | `for`, `while`, `do-while` loops, printing number and star patterns       |
| 21 – 30 | Arrays & Basic Operations          | Creating arrays, finding max/min, basic array manipulations               |
| 31 – 40 | Decision Structures & Combinations | `if`, `else if`, `switch`, logical conditions, combination-style problems |

---

## Technologies Used

* C#
* .NET standard libraries (System, System.Collections, etc.)
* Basic algorithm logic

---

## Example Exercise Format

```csharp
// Example: Sum of two numbers
Console.Write("Enter first number: ");
int a = int.Parse(Console.ReadLine());
Console.Write("Enter second number: ");
int b = int.Parse(Console.ReadLine());
int sum = a + b;
Console.WriteLine("Sum: " + sum);
```
